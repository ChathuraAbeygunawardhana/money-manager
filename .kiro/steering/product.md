# Product Overview

Lankachat is a real-time chat application with role-based access control. Users can join chatrooms and send messages, while admins have additional privileges to create chatrooms, manage users, and view analytics.

## Core Features

- User authentication with secure password hashing
- Real-time messaging with 2-second polling
- Chatroom creation (admin only) and joining
- Role-based access control (admin/user roles)
- Admin analytics dashboard with user and chatroom statistics
- User management interface (CRUD operations for admins)
- Auto-membership for admins in all chatrooms

## User Roles

- **Admin**: Can create chatrooms, manage users, view analytics, and access all chatrooms automatically
- **User**: Can join existing chatrooms and send messages
